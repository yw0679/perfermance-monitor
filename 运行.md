## 压测
```
# stress压测cpu
# 安装
sudo apt-get install -y stress
# 压满所有cpu
stress --cpu "$(nproc)" --timeout 60s

# iperf3压测网络
# 对端启动服务端
iperf3 -s
# 本机压发送方向
iperf3 -c 192.168.63.1 -t 60 -P 4
# 本机压接受方向
iperf3 -c 192.168.63.1 -t 60 -P 4 -R
# 双向
iperf3 -c 192.168.63.1 -t 30 -P 4 --bidir

```
