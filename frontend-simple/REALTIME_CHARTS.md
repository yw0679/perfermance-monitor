# 🚀 实时曲线图监控界面

## ✅ 已升级！现在支持实时曲线图

前端已改造为**实时动态曲线图**模式，每1秒自动更新。

## 📊 新功能特性

### 🎨 实时曲线图
- **CPU 使用率趋势** - 红色曲线，实时显示处理器负载
- **内存使用率趋势** - 蓝色曲线，监控内存占用
- **网络流量趋势** - 绿色(发送) + 紫色(接收) 双曲线
- **系统负载趋势** - 橙色曲线，显示系统平均负载

### ⚡ 实时特性
- ✅ **1秒刷新** - 每秒自动更新数据
- ✅ **60秒历史** - 显示最近60个数据点（1分钟滚动窗口）
- ✅ **平滑动画** - 曲线平滑过渡，视觉效果流畅
- ✅ **暗色主题** - GitHub 风格深色界面，护眼舒适

## 📈 数据更新机制

### 时序图
```
每1秒:
浏览器 → API (/api/servers) → MySQL → 返回最新数据 → 更新曲线图
   ↓
追加新数据点到曲线
   ↓
保持最近60个点
   ↓
平滑动画渲染
```

### 数据流
1. **JavaScript 定时器**: `setInterval(fetchRealtimeData, 1000)`
2. **API 查询**: 获取服务器最新快照
3. **数据追加**: 添加到 `dataHistory` 数组
4. **窗口滚动**: 超过60个点自动删除旧数据
5. **图表更新**: `chart.update('none')` 无动画快速刷新

## 🎨 技术实现

### Chart.js 配置
```javascript
// 平滑曲线
tension: 0.4

// 填充区域
fill: true

// 快速更新（无动画）
chart.update('none')

// 滚动窗口
```

## 🚀 实时数据要求

### 获取实时数据流

确保 worker 正在运行以获得1秒更新：

```bash
# 终端1: 启动 Manager
./build/manager/manager 0.0.0.0:50051

# 终端2: 启动 Worker（每1秒上报）
sudo ./build/worker/worker 127.0.0.1:50051 1
```

### 数据验证

检查数据是否实时更新：
```bash
# 监控数据库最新记录
watch -n 1 "mysql -h127.0.0.1 -umonitor -pmonitor123 monitor_db -e 'SELECT server_name, cpu_percent, timestamp FROM server_performance ORDER BY timestamp DESC LIMIT 1;'"
```

## 💡 使用技巧

### 1. 监控压力测试
在压测时观察曲线变化：
```bash
# 压 CPU
stress --cpu $(nproc) --timeout 60s
# 观察红色CPU曲线飙升

# 压网络
iperf3 -c 192.168.1.20 -t 60 -P 4
# 观察绿色/紫色网络曲线上升
```

### 2. 多窗口对比
- 打开多个浏览器窗口
- 每个窗口选择不同服务器
- 并排对比性能

### 3. 导出图表（浏览器功能）
- 右键点击图表 → 保存图片
- 用于报告和分析

## 🛠️ 技术架构

```
┌─────────────────────────────────────────────────┐
│              浏览器 (JavaScript)                 │
│  - Chart.js 绘制曲线                            │
│  - 1秒定时器 (setInterval)                      │
│  - 60点滚动窗口                                 │
└─────────────────────────────────────────────────┘
                     ↓ HTTP GET /api/servers
┌─────────────────────────────────────────────────┐
│          Flask REST API (Python)                │
│  - 查询 MySQL                                   │
│  - 返回 JSON                                    │
└─────────────────────────────────────────────────┘
                     ↓ SQL Query
┌─────────────────────────────────────────────────┐
│              MySQL 数据库                        │
│  - server_performance 表                        │
│  - 最新记录查询                                 │
└─────────────────────────────────────────────────┘
                     ↑ INSERT
┌─────────────────────────────────────────────────┐
│              Manager (C++)                      │
│  - 接收 gRPC                                    │
│  - 写入数据库                                   │
└─────────────────────────────────────────────────┘
                     ↑ gRPC
┌─────────────────────────────────────────────────┐
│              Worker (C++)                       │
│  - 每1秒采集                                    │
│  - eBPF + 内核模块                              │
└─────────────────────────────────────────────────┘
```
